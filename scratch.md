## 1.1.4

**<p> Вычислим искомую вероятность методом Монте-Карло: </p>**
**<p> Пусть X,Y - время прибытия первого и второго парохода соответственно (независимые случайные величины) </p>**
**<p> Тогда вероятностное пространство для задачи можно представить как множество точек (X,Y) в квадрате [0,24]\*[0,24] </p>**
**<p> Найдем вероятность противоположного события - т.е. один пароход не ждет другого = P0 </p>**
**<p> 1. Первый уходит до второго: P1 = $P(X + 1 \leq Y)$ = площадь треугольника 23\*23 / площадь квадрата 24\*24 = 264.5 / 576 (можно вычислить через интеграл)</p>**
**<p> 2. Второй уходит до первого: P1 = $P(Y + 2 \leq X)$ = площадь треугольника 22\*22 / площадь квадрата 24\*24 = 242 / 576 (можно вычислить через интеграл)</p>**
**<p> 3. Итоговая вероятность: P0 = P1 + P2 = 506.5 / 576 </p>**
**<p> Осталось найти вероятность события противоположного P0 </p>**
**<p> Ответ: 1 - P0 = 69.5 / 576 = 0.1207</p>**
